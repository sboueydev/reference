    Resources

    This is my personal list of resources inspired by various awesome lists on GitHub.

    Note: ✓ denotes something I have personally used before

    Preprocessors

    HTML Preprocessors

    • Pug ✓ - Clean and whitespace sensitive html preprocessors formerly known as Jade.

    Frameworks

    • W3.CSS ✓ - This is a CSS framework by w3 schools. It has several gorgeous example templates which display wonderfully on the my mobile device.

    Update: I tested this framework out recently and I disliked the extra markup it added. I have found this with Bootstrap as well so it is not per say the fault of the framework, just personal preference.

    Static Sites

    Add-ons

    • Staticman - Tool to add user-generated content to static sites.

    Wordpress

    Wordpress Frameworks

    • TypeRocket ✓ - This is free and it looks really cool. It has a pagebuilder and custom fields. Setup requires Composer. According to the website it's *"like Advanced Custom Fields + Laravel + Magic in WordPress"

    Update: I have now tested this and it's pretty darn cool.

    • Unyson - Free drag and drop wordpress framework.
    • Gantry - Free framework. Includes drag and drop capabilities and works with Grav and Joomla as well as Wordpress.
